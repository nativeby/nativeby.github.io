Admin 后台管理系统

后台管理平台

我们在开发系统的时候通常会涉及到后台管理系统，现在很多后台管理系统都以 -admin 来命名，比如：

  1. go-admin, 很显然这个是用 golang 语言开发的后台管理系统
  2. react-admin，很显然这个前端是用 react 语言开发的后台管理系统，至于后端语言用什么，可以随意了，python、 nodejs 、 php、 golang 等都行
  3. vue-admin，很显然这个前端是用 vue 语言开发的后台管理系统
