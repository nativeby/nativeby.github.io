Python Summary

内置数据类型

  1. list — 列表 可变有序集合，可以随时添加和删除其中的元素。

    • 表达
    • 操作
      • 下标 — 0是第一个元素、-1是最后一个元素
      • len()
      • append() — 往list中追加元素到末尾
      • insert(x,y) — 把元素插入到指定的位置
      • pop(x) — 删除list末尾的元素或指定位置的元素

  2. tuple — 元祖 不可变有序集合，tuple和list非常类似，但是tuple一旦初始化就不能修改。

    • 表达
    • 操作 同 list，但是不再有增加删除元素操作

  3. 字典 — dict

  4. 集合 — set
